Detailed Description

Examples include: A console. A UI. WS2811 protocol. DMX protocol. Data pin. It contains a list of Views that feed it the data to "display." A Format mainly calls the handles for all it's known Views in every loop

◆ update_views()

void Format::update_views ( )

If a view is defined as stale, call on_refresh, on_paint, and on_display for the stale View. Stale is defined as different things for different devices. For System_View, it's that there's a new message on the System_Message stack. For a device, it's that the state has changed in some meaningful way.
